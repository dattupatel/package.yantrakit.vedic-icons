# Vedic Icons - Icon Set

## Initial Icons (from Yantrakit Clock)

These 11 icons exist in the clock project and will be the starting point:

| # | Icon Name | Class | Description |
|---|-----------|-------|-------------|
| 1 | Yantra | `vi-yantra` | Sri Yantra sacred geometry — outer circle with intersecting triangles, center bindu |
| 2 | Surya | `vi-surya` | Sun symbol — central circle with 8 rays and inner dot |
| 3 | Scroll | `vi-scroll` | Ancient manuscript — scroll rolls on sides with text lines |
| 4 | Namaste | `vi-namaste` | Prayer hands — two hands in vertical alignment with lotus hint |
| 5 | Temple | `vi-temple` | Gopuram/temple — multi-tiered architecture with shikhara and kalash |
| 6 | Coin | `vi-coin` | Ancient coin — dual circles with simplified Om in center |
| 7 | Panchang | `vi-panchang` | Calendar grid — calendar frame with decorative top and moon phase |

| 9 | Sangha | `vi-sangha` | Community — three figures representing unity/groups |

## Planned Categories

### 1. Sacred Geometry
- Sri Yantra, mandala, kolam patterns, rangoli motifs, mehndi patterns

### 2. Spiritual Symbols
- Om, lotus, diya, rudraksha, trishul, damru, shankh (conch), kalash, swastik, tilak, bindi, kumkum

### 3. Architecture
- Gopuram, stupa, jali patterns, arches, haveli, minaret, dome, pillar

### 4. Cultural
- Rangoli, mehndi, namaste, bindi, turban, sari, dhoti, bangle, nose ring

### 5. Nature (Indian context)
- Banyan tree, peacock, elephant, cow, tiger, cobra, parrot, mango, neem, tulsi

### 6. Festivals
- Diya, firework, lantern, toran, garland, puja thali, haldi, mehendi

### 7. Music & Dance
- Tabla, sitar, veena, mridangam, bharatanatyam pose, mudra hand gestures

### 8. Food & Drink
- Chai, thali, samosa, ladoo, naan, masala dabba

### 9. General Purpose (Vedic aesthetic)
- Arrows, checkmarks, close, menu, search, settings, user, home, heart, star, bell, lock, mail, phone, calendar, clock, download, upload, edit, delete, share, bookmark, filter, sort, grid, list, plus, minus, chevrons, external link, info, warning, error, success

The general purpose icons will follow the same Vedic design language — rounded, organic strokes inspired by Devanagari letterforms — so users don't need a second icon library for standard UI elements.

## Target Count

- **v1 (free)**: 200-400 icons
- **v2 (pro)**: 1000+ icons across multiple styles (light, regular, solid, duotone)
